Installing VS Code Extensions

VS Code extensions allow you to add features like code suggestions, function signature help, debugging tools, and build support, turning VS Code into a more capable IDE. Below is an example of how to install an extension.

Adding an HTML Viewer to VS Code

This guide explains how to set up a live HTML preview in VS Code to view documentation, such as locally saved HTML files, directly within the editor.

Why Use an HTML Viewer in VS Code?

Instead of opening HTML files in a separate browser, a live preview lets you see rendered content alongside your code, updating in real-time as you edit. This is ideal for reviewing documentation or prototyping web pages.

Recommended Tool: Live Preview Extension

The Live Preview extension by Microsoft is lightweight, actively maintained, and perfect for rendering HTML files within VS Code.

Steps to Set Up

  1. Install the Live Preview Extension
    • In VS Code, press Ctrl+Shift+X (or Cmd+Shift+X on macOS) to open the Extensions view.
    • Search for Live Preview.
    • Click Install on the extension by Microsoft (ms-vscode.live-server).
  2. Open Your HTML File

    Open the HTML file you want to preview (e.g., cmake-vscode-docs.html) in the VS Code editor.

  3. Start the Preview
    • Right-click anywhere in the HTML file.
    • Select Show Preview from the context menu.
    • Alternatively, press Ctrl+Shift+P (or Cmd+Shift+P), type Live Preview: Show Preview, and hit Enter.

    A preview pane will open within VS Code, rendering the HTML content.

  4. Enable Live Updates (Optional)

    The preview updates automatically as you edit the HTML file, thanks to an embedded server running at http://127.0.0.1
