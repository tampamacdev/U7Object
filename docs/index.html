Documentation

This is a collection of guides to build and modify the CMake sample code project that demonstrate how to create a WebSocket service. The majority of the example code resides in the Vala project contained in the vala/ directory. A large portion of the HTML documentation was generated by ChatGPT and Grok and copied into files here. There's even documentation on how to prompt an LLM to generate useful HTML versions of AI generated responses.
